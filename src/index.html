Kyubot

Deployed URL       GitHub

A web dashboard which allows a user to control a Sphero robot via Bluetooth in the browser. Users can control the direction and velocity of the robot via a virtual joystick, and change the colors of the Shpero's LEDs. Due to the newness of the browser-based Bluetooth functionality, this app is currently only supported in a Chrome OS environment.

Smashtag!

Deployed URL       GitHub

This app allows a user to see a real-time comparison of tweets. The user inputs up to three hashtags, and the app displays a running total of tweets, and a graph representing percentages of topics by total tweet volume. This is all made possible by using the Twitter Streaming API.
